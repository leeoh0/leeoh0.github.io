Macro Excel Registration Form

Designing a structured and user-friendly interface is the first step in creating and automating a printable registration form using Microsoft Excel and VBA (Visual Basic for Applications). The Excel sheet have input fields like sr code, name, and other necessary details, all clearly labeled and organized logically. Utilizing VBA, a user form with text boxes, dropdown lists, and buttons were created to efficiently capture user data. A "Submit" button connected to a VBA macro facilitates the transfer of data from the form to a database sheet for orderly storage. Moreover, a VBA script was used to format the collected data into a well-structured printable template with consistent font styles, spacing, and alignment. By incorporating a "Print" button, users can easily generate and print the completed registration form with just one click, saving time, reducing errors, and improving the overall efficiency of the registration process.

Flight Booking System Console Application

Creating a flight booking system through a console program involves developing a well-organized application that allows users to efficiently carry out CRUD (Create, Read, Update, Delete) functions. Users could add new flight reservations, view existing bookings, modify details like flight time or passenger information, and delete bookings. The system utilized arrays as the primary data structure to store flight and passenger information for easy access. Branching was used to manage user input and direct them through different options, such as booking, updating, or canceling a flight. The code has been modularized with functions dedicated to each operation (e.g., createBooking(), viewBookings(), updateBooking(), deleteBooking()) to enhance readability and reusability. Additionally, error handling and input validation was integrated to ensure a smooth and user-friendly experience.

Simple Café Ordering System

This C++ program was created to stimulate a standard café ordering system in a digital setting. It starts by displaying a well-structured menu containing variety of items and their prices, making it easy for users to choose and navigate through the selections. When the menu is shown, users are asked to enter their preferred item number and quantity. Each valid selection is processed to determine the cost, which is then added to the running total. Additionally, users can keep adding items until they decide to exit by entering 0. Errors like entering incorrect numbers or negative quantities are handled by showing error messages and asking for correct input. As the ordering session continues, the system accurately calculates the total cost of the order and presents it to the user. This careful focus on detail offers a seamless and engaging experience with input verification for reliable performance.
